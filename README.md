# beta-mahjong-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容各自独立，不针对任何单个域名或具体网站。

## 项目简介

本仓库主要存放与麻将相关的一些辅助页面或工具页面。各页面均为自包含的 HTML 文件，可直接在浏览器中打开使用，无需额外依赖。

## 目录说明

```
.
├── README.md          # 本说明文件
├── page1.html         # 示例页面（实际文件名以仓库内容为准）
├── page2.html
└── ...
```

仓库根目录下直接存放各 HTML 页面文件，文件命名清晰，便于索引。

## 页面归档说明

- 每个 HTML 文件均为独立页面，不依赖于其他文件或外部资源（除标准网络字体、CDN 库等通用资源外）。
- 页面内容可能涉及麻将规则、牌型计算、练习工具等，具体用途请查看各文件内的注释或页面标题。
- 本仓库不提供在线演示环境，如需浏览页面，请将仓库克隆到本地，或通过 GitHub Pages 等功能自行部署。

## 维护说明

- 本仓库不定期更新，可能新增、修改或删除页面。
- 欢迎通过 Issue 提交建议或报告问题。
- 不接受直接向本仓库提交包含具体域名、营销信息或诱导点击内容的 Pull Request。
- 所有页面内容仅供学习与参考，使用前请自行判断其适用性。
